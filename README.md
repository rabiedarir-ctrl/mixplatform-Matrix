# 🌐 Mix Platform – منصة رقمية موحدة

**Mix Platform** هي منصة رقمية متكاملة لتشغيل تطبيقات مختلفة على الهواتف والأجهزة، مع دعم **PWA** وOffline وBackend متكامل.

---

## 🚀 الوحدات الرئيسية

| الوحدة | أيقونة | وصف | معاينة |
|--------|--------|-----|--------|
| Social | 💬 | نشر ومتابعة التحديثات، التعليقات، الصور، البث المباشر، دمج AI | [عرض](frontend/index.html#social) |
| Store | 🛒 | إدارة المتجر الرقمي، عرض عناصر للبيع | [عرض](frontend/index.html#store) |
| Wallet | 💰 | إدارة العملات الرقمية والمعاملات المالية | [عرض](frontend/index.html#wallet) |
| Games | 🎮 | حزمة ألعاب متصلة بالبيتكوين | [عرض](frontend/index.html#games) |
| Metaverse | 🌐 | عالم افتراضي ثلاثي الأبعاد | [عرض](frontend/index.html#metaverse) |
| Matrix | 🧩 | دردشة وتعاون لحظي بين المستخدمين | [عرض](frontend/index.html#matrix) |
| GPS | 🛰️ | تتبع الموقع، GeoFence، خدمات الطوارئ | [عرض](frontend/index.html#gps) |
| AI | 🧠 | توصيات ذكية وتحليلات بيانات | – |
